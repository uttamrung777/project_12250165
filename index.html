Practical Test Information

The Bachelor of Information and Communication Technology Practical Test is designed to evaluate your technical skills and practical understanding of core ICT concepts.

Test Overview

This practical examination assesses your competency in various ICT domains including:

  • Web Development: HTML5, CSS3, JavaScript
  • Programming: Python, Java, or C#
  • Database Management: SQL queries and database design
  • Networking: Basic network configuration and troubleshooting
  • System Analysis: Problem-solving and design thinking

Important Instructions

All students must adhere to the following guidelines:

  • Ensure you have a stable internet connection during the test
  • Save your work frequently
  • Submit all required files before the deadline
  • No collaboration with other students is permitted
  • Cite all external resources used

Test Requirements

To successfully complete the practical test, you will need:

  • Code editor (VS Code, Sublime Text, or similar)
  • Web browser (Chrome, Firefox, or Edge)
  • Database management tool (phpMyAdmin, MySQL Workbench)
  • File compression tool (WinZip, 7-Zip, or similar)

Submission Guidelines

Please note the following submission requirements:

  • All code files must be properly commented
  • Include a README file with setup instructions
  • Compress all files into a single ZIP folder
  • Name your submission file: StudentID_BICT_Practical.zip
  • Late submissions will incur penalties
